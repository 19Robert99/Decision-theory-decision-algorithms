Редактирование переменной

Для редактирования переменной необходимо выбрать ее в дереве проекта, после этого отобразится панель редактирования переменной, показанная на рисунке. В поле "Назавание" вводится название переменной.

Вид переменной выбирается из списка, возможные значения: входная, промежуточная, выходная. Значение входной переменной вводится пользователем в процессе консультации. Входная переменная не может содержаться в правой части правила и не может быть целью консультации. Промежуточная переменная может содержаться как в левой части правила, так и в правой. Выходная переменная может содержаться только в правой части правила и может являться целью консультации.

Тип переменной выбирается из списка, возможные значения: дискретная, числовая. Числовая переменная может принимать действительные числовые значения. Для дискретной переменной задается список возможных значений. Элементы списока значений редактируются с помощью кнопок: добавить, редактировать, удалить, вверх, вниз.

Для входных переменных в поле "Вопрос" может быть введен вопрос, который будет задаваться пользователю при вводе значения переменной.
